# Biblioteca

Biblioteca que encapsula requisições submetidas para a RNDS por meio de uma interface em Java.

### Maven

```xml
<dependency>
  <groupId>com.github.kyriosdata</groupId>
  <artifactId>rnds-lib</artifactId>
  <version>0.0.3</version>
</dependency>
```
